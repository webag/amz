Методы для открытия закрытия модальных окон:

####Конец триала:
```javascript
open_trial_modal()
close_trial_modal()
```

####Остальные окна

вызываются так:
```javascript
$('#айдимодалки или ее класс').agmodal('open');
```

закрываюстя так:
```javascript
$('#айдимодалки или ее класс').agmodal('close');
```

####Метод для построения ширин столбцов
```javascript
columns_sizes()
```
